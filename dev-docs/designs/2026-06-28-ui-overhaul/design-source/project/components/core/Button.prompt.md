Solid-cyan action button with terminal-dense sizing; use `ghost`/`icon` variants for toolbars and `destructive` for irreversible actions.

```jsx
<Button variant="primary" size="md">Create task</Button>
<Button variant="ghost" size="icon"><PlusIcon /></Button>
<Button variant="destructive" size="sm">Delete</Button>
```

Variants: `primary` (default), `secondary`, `outline`, `ghost`, `destructive`, `link`.
Sizes: `xs`, `sm`, `md` (default), `lg`, `icon`.
