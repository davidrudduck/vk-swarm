# Task: Run full verification

## Goal
Verify all changes work together.

## Context
This is the final verification task. All previous tasks should be complete before running this.

## Steps
1. Run Rust build
2. Run Rust tests
3. Generate TypeScript types
4. Run frontend lint
5. Run frontend type check
6. Run combined check command

## Files to Modify
None - verification only

## Code

Run these commands from the project root:

### 1. Rust build
```bash
cargo build --workspace
```
Expected: Compiles successfully with no errors

### 2. Rust tests
```bash
cargo test --workspace
```
Expected: All tests pass

### 3. Generate TypeScript types
```bash
npm run generate-types
```
Expected: Types generated successfully

### 4. Frontend lint
```bash
cd frontend && npm run lint
```
Expected: No lint errors

### 5. Frontend type check
```bash
cd frontend && npx tsc --noEmit
```
Expected: No type errors

### 6. Combined check (optional)
```bash
npm run check
```
Expected: All checks pass

### 7. Verify MCP tools (manual)

Start the dev server:
```bash
pnpm run dev
```

Check that the new MCP tools appear in the server instructions. The tools should include:
- `list_templates`
- `get_template`
- `send_follow_up`
- `send_template`
- `get_last_message`

The `start_task_attempt` tool should now have a `target_node_id` parameter.

The `get_task_attempt_status` tool should return `task_title`, `is_running`, and `has_failed` fields.

### 8. Verify templates API (manual)

With the dev server running, test the templates endpoint:
```bash
curl http://localhost:BACKEND_PORT/api/templates/all
```

Expected: Returns JSON array with system templates (at minimum).
