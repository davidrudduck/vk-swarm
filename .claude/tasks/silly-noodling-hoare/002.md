# Task: Sort Configuration Dropdown in AgentSettings

## Goal
Add alphabetical sorting to the configuration/variant dropdown on the Settings → Agents page.

## Context
The Settings → Agents page (`AgentSettings.tsx`) has a Configuration dropdown that shows variant names like "DEFAULT", "fast", "thorough", etc. for the selected executor type. This dropdown should display options in alphabetical order for consistency.

Task 001 already added sorting to the Executor Type dropdown in the same file. This task adds sorting to the Configuration dropdown.

## Steps
1. Open `frontend/src/pages/settings/AgentSettings.tsx`
2. Find lines 466-469 where the configuration dropdown items are mapped
3. Add `.sort()` after the `Object.keys(...)` call and before `.map()`

## Files to Modify
- `frontend/src/pages/settings/AgentSettings.tsx` - lines 466-469: add `.sort()` to configuration mapping

## Code

Find this code block (around lines 465-477):
```typescript
                      <SelectContent>
                        {Object.keys(
                          localParsedProfiles.executors[selectedExecutorType] ||
                            {}
                        ).map((configuration) => (
                          <SelectItem key={configuration} value={configuration}>
                            {configuration}
                          </SelectItem>
                        ))}
                        <SelectItem value="__create__">
                          {t('settings.agents.editor.createNew')}
                        </SelectItem>
                      </SelectContent>
```

Change it to (add `.sort()` after the closing `)` of `Object.keys(...)` and before `.map`):
```typescript
                      <SelectContent>
                        {Object.keys(
                          localParsedProfiles.executors[selectedExecutorType] ||
                            {}
                        ).sort().map((configuration) => (
                          <SelectItem key={configuration} value={configuration}>
                            {configuration}
                          </SelectItem>
                        ))}
                        <SelectItem value="__create__">
                          {t('settings.agents.editor.createNew')}
                        </SelectItem>
                      </SelectContent>
```

The change is on line 469:
- FROM: `).map((configuration) => (`
- TO: `).sort().map((configuration) => (`
