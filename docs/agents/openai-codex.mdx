---
title: "OpenAI Codex"
description: "Set up OpenAI Codex integration"
icon: "terminal"
---

<Steps>
<Step title="Run OpenAI Codex">
  ```bash
  npx -y @openai/codex
  ```
</Step>

<Step title="Follow the login instructions">
  Complete the authentication flow as prompted.

  Alternatively, Codex can be used via the OpenAI API by setting the `OPENAI_API_KEY` environment variable.
</Step>

<Step title="Start Vibe Kanban">
  Once authenticated, launch Vibe Kanban:

  ```bash
  npx vibe-kanban
  ```

  You can now select OpenAI Codex when creating task attempts. Native Codex collaboration modes are discovered from the runtime at startup, and unsupported modes fall back to standard execution automatically.
</Step>
</Steps>
